## Skills Overview

### Brainstorm (Development Methodology)
- **Superpowers** - Complete software development methodology from [obra/superpowers](https://github.com/obra/superpowers) (260k stars) covering:
  - Brainstorming & design refinement
  - Implementation planning
  - Test-driven development (TDD)
  - Subagent-driven workflows
  - Systematic debugging
  - Code review
  - Git worktree management

### DevTools
- **API Tester** - Test HTTP APIs using Chrome DevTools MCP
- **Design Master** - Comprehensive design skills for UI/UX, branding, slides, and visual content

### Mobile
- **Android** - Skills from Google's [android/skills](https://github.com/android/skills) repository covering:
  - Jetpack Compose & theming
  - CameraX development
  - Navigation Component 3
  - Testing & performance
  - Wear OS & XR
  - Security & identity
  - Play publishing

## Usage

Each skill directory contains a `SKILL.md` file with detailed instructions for the AI agent.
